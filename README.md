# education folder (study kit)

This folder is **optional study material** for building the Cafe POS assignment. Your **graded code** lives **outside** this folder (ASP.NET Core MVC project at the repo root or wherever you create the blank solution).

## remove this folder

Delete the entire `education/` directory. Nothing in the app references it.

## two ways to read this (pick one)

### path a — you have ~3 days until due

Read in this order:

1. [CRITICAL_PATH_3DAY.md](CRITICAL_PATH_3DAY.md) — what to build each day and what to cut if late  
2. [PRD.md](PRD.md) — acceptance tests you can tick off  
3. [WORKFLOW_MAP.md](WORKFLOW_MAP.md) — how sample interactions map to screens  
4. [DATA_MODEL.md](DATA_MODEL.md) — tables and relationships (matches `db/setup.sql`)  
5. [templates/PROMPTS.md](templates/PROMPTS.md) + [AI_GUIDE.md](AI_GUIDE.md) — use AI as tutor, not autopilot  
6. [STYLE_COMMENTS.md](STYLE_COMMENTS.md) — how you want comments to look  

Build the app in parallel. Do **not** wait until every doc is read.

### path b — you have time to go deep

1. [GLOSSARY.md](GLOSSARY.md)  
2. [ARCHITECTURE.md](ARCHITECTURE.md)  
3. [DATA_MODEL.md](DATA_MODEL.md)  
4. [PHASES/](PHASES/) in numeric order  
5. [SOP.md](SOP.md) for daily rhythm
